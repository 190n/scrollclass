Scrollclass example
===================

This example uses [webpack](https://webpack.js.org/) to import Scrollclass as an
ES6 module.

[See the live version here.](https://190n.github.io/scrollclass/example/example.html)

Building
--------

Install dependencies (just Scrollclass):

```bash
$ npm install
```

Run Webpack (must be installed):

```bash
$ webpack
```
